# Безопасно ли это?

## Безопасно ли использовать моды?
**Да, использование косметических модов безопасно.** Valve допускает использование модификаций, которые изменяют только локальные файлы клиента (модели, текстуры, иконки, звуковые эффекты) - об этом написано [здесь](https://dota2.fandom.com/ru/wiki/%D0%91%D0%B0%D0%BD#%D0%98%D1%81%D0%BA%D0%BB%D1%8E%D1%87%D0%B5%D0%BD%D0%B8%D1%8F).

::: tip Примечание
Косметические модификации влияют исключительно на ваш локальный клиент. Они не дают игровых преимуществ и не меняют логику игры на сервере.
:::

## Сравнение с другими чейнджерами
Существуют различные подходы к модификации Dota 2. Понимание того, как работают инструменты, поможет вам сделать осознанный выбор в пользу безопасности.

| Тип модификации | Метод работы | Риск |
| :--- | :--- | :--- |
| **Ресурсы .vpk (наш метод)** | Использование стандартной системы загрузки ресурсов игры (только vpk) | Минимальный |
| Модификация .vpk и сигнатур | Модификация файлов dota.signatures и gameinfo | Низкий |
| Инъекция в процесс | Внедрение кода напрямую в память игры | Высокий |

::: tip Примечание
Несмотря на то, что популярные чейнджеры работают через модификацию сигнатур, за 8-9 лет использования не было зафиксировано случаев блокировок.
:::

### Особенности нашего подхода
Большинство сторонних чейнджеров для работы с предметами используют модификацию файла `items_game.txt`. Это требует манипуляции с сигнатурами, что Valve может классифицировать как вмешательство.

**Преимущества наших модов:**
* Мы не затрагиваем исполняемые файлы игры и не меняем сигнатуры.
* Наши моды не перестают работать после выхода обновлений Dota 2, так как они не привязаны к жестким адресам.
* Модификации загружаются аналогично официальным пакетам локализации (озвучки), используя vpk.
