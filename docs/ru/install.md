# Как установить моды?

Существует несколько способов установки модов с сайта. Выберите наиболее подходящий для вас вариант:

1. [Ручная установка каждого мода отдельно](#manual-single)
2. [Ручная установка через корзину (без VPKMerge)](#manual-cart-no-merge)
3. [Ручная установка через корзину (с VPKMerge)](#manual-cart-merge)
4. [Автоматическая установка через корзину (с VPKMerge)](#auto-install-merge)

---

## 1. Ручная установка каждого мода отдельно {#manual-single}

Этот способ подходит, если вам нужно установить всего один или несколько модов.

1. На сайте нажмите на интересующий вас мод, чтобы скачать его. Проделайте это со всеми необходимыми модами.
2. Откройте **Steam**, нажмите правой кнопкой мыши (ПКМ) по **Dota 2** → **Управление** → **Просмотреть локальные файлы**.
3. Перейдите в директорию `game`.
4. Откройте папку, соответствующую языку вашей игры (например, `dota_russian` для русского языка или `dota_123` для английского (ее нужно создать)).
5. Перенесите все скачанные файлы модов в эту папку.

::: danger ВАЖНО
Названия файлов модов должны быть строго в диапазоне от `pak01_dir.vpk` до `pak99_dir.vpk`. В противном случае игра может вылететь или моды просто не будут работать.
:::

## 2. Ручная установка через корзину (без VPKMerge) {#manual-cart-no-merge}

Удобно, если вы хотите скачать пак модов, без использования VPKMerge.

1. На сайте наведите курсор на карточку мода и нажмите на **плюс (+)** в правом верхнем углу. Добавьте таким образом все нужные моды.
2. Откройте корзину и нажмите кнопку **Download**.
3. Дождитесь окончания пака и скачивания архива.
4. Распакуйте скачанный архив и перенесите папку из него на рабочий стол.
5. Откройте **Steam**, нажмите ПКМ по **Dota 2** → **Управление** → **Просмотреть локальные файлы**.
6. Перейдите в директорию `game`, затем откройте папку языка игры (например, `dota_russian`).
7. Перенесите все файлы модов из распакованной папки `mods` в папку с языком игры.

::: warning ОБРАТИТЕ ВНИМАНИЕ
У вас могут появиться файлы с названиями вида `!pak01_dir.vpk` или `pak99_dir_1.vpk`. Их необходимо переименовать:
* Файлы, содержащие знак `!`, имеют наивысший приоритет. Переименуйте их так, чтобы цифра в названии была меньше, чем у остальных файлов (например, `pak01_dir.vpk`).
* Файлы с именами вроде `pak99_dir.vpk`, `pak99_dir_1.vpk` и т.д. необходимо объединить в один общий файл с помощью программы **VPKTool**. Итоговый файл должен называться `pak99_dir.vpk`.
:::

## 3. Ручная установка через корзину (используя VPKMerge) {#manual-cart-merge}

Использование VPKMerge позволяет автоматически объединить моды в несколько файлов, избегая конфликтов и ручного переименования множества файлов.

1. На сайте наведите курсор на карточку мода и нажмите на **плюс (+)** в правом верхнем углу. Сформируйте ваш список модов.
2. Перейдите в корзину и нажмите кнопку **Download**.
3. Дождитесь завершения загрузки архива.
4. Распакуйте полученный архив на рабочий стол.
5. Перейдите в папку `mods` внутри распакованного архива и запустите **VPKMerge**.
6. Программа автоматически объединит файлы и создаст архивы вида `pak10_dir.vpk`, `pak10_000.vpk`, `pak10_001.vpk` и т.д.
7. Откройте **Steam**, нажмите ПКМ по **Dota 2** → **Управление** → **Просмотреть локальные файлы**.
8. Перейдите в директорию `game` → папка вашего языка (например, `dota_russian`).
9. Перенесите **все** созданные файлы, название которых начинается на `pak10` (включая `.vpk` и нумерованные части), из папки `mods` в папку языка игры.

## 4. Автоматическая установка через корзину (используя VPKMerge) {#auto-install-merge}

Самый простой и быстрый способ установки, который сделает всё за вас.

1. В настройках сайта предварительно выберите язык вашей игры и систему.
2. Наведите курсор на карточку мода и нажмите на **плюс (+)** в правом верхнем углу для каждого нужного мода.
3. Откройте корзину и нажмите кнопку **Download**.
4. Дождитесь скачивания архива и распакуйте его на рабочий стол.
5. Запустите исполняемый файл **Auto-Install** из распакованной папки.
6. Дождитесь окончания процесса установки
