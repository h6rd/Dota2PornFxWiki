# Решение проблем

## Проблемы с ландшафтами
Ландшафты делятся на два типа, каждый из которых требует своего способа установки:

* **Кастомные:** Устанавливаются как обычные моды путем переноса `.vpk` файла в папку языка.
* **Стандартные (из Dota 2):** Устанавливаются путем переноса папки `maps`, содержащей `dota.vpk`, в папку языка.

::: warning ВАЖНО
* **Не перемещайте и не переименовывайте** файл `dota.vpk` внутри папки `maps`.
* Стандартные ландшафты отображаются **только в матчах**. В режиме демо они не работают.
* Обязательно **отключите ландшафт Dota+** в настройках игры, иначе он будет конфликтовать с модом.
:::

## Нет изменений или не работает мод
Если после установки моды не отображаются, выполните следующие шаги:

### 1. Подготовка папки
1. В Steam: ПКМ по Dota 2 → **Управление** → **Просмотреть локальные файлы**.
2. Перейдите в директорию `game`.
3. Откройте папку языка игры (например, `dota_russian` для русского языка).

### 2. Проверка имен файлов
Перенесите `.vpk` файлы в папку языка. Убедитесь, что они соответствуют правилам именования:
* Допустимый диапазон: `pak01_dir.vpk` — `pak99_dir.vpk`.
* **Запрещено:** использование произвольных названий (например, `mod.vpk`, `pak120_dir.vpk`) и наличие спецсимволов (восклицательных знаков и т.д.).
* Если у вас более 100 модов и номера закончились, воспользуйтесь утилитой [VPKMerge](https://h6rd.github.io/Dota2PornFxWeb/?category=tools&mod=VPKMerge+-+Combine+VPKs).

### 3. Параметры запуска
Убедитесь, что в параметрах запуска Dota 2 установлена корректная команда:
1. В Steam: ПКМ по Dota 2 → **Свойства** → **Общие**.
2. В поле «Параметры запуска» впишите: `-language russian`
3. Убедитесь, что эта команда стоит первой и является единственной (если вы не используете другие обязательные параметры).

## Ошибка VAC
Если при запуске игры или поиске матча появляется сообщение об ошибке **VAC**, не ссыте. Это баг доты, который не является следствием использования модов.

**Как исправить:**
1. Откройте **Steam**.
2. Нажмите ПКМ по Dota 2 → **Свойства**.
3. Перейдите на вкладку **Установленные файлы**.
4. Нажмите **Проверить целостность файлов игры**.

После завершения проверки перезапустите игру.


## Загрузка на сайте зависает

### Если загрузка останавливается, попробуйте:
* Использовать VPN
* Использовать одно из зеркал сайта: [**Netlify**](https://d2pfx.netlify.app/), [**Vercel**](https://d2pfx.vercel.app/), [**Onrender**](https://d2pfx.onrender.com/), [**Codeberg**](https://hrdq.codeberg.page/Dota2PornFxWeb/)
* Использовать zapret (если вы в РФ) c добавлением в файл `list-general.txt`:
```text
github.com
github.io
raw.githubusercontent.com
assets-cdn.github.com
githubassets.com
githubusercontent.com
```

### Если это не помогло:
1. Обновите страницу
2. Откройте консоль сочетанием `Ctrl + Shift + J`
3. Начните скачивание через корзину
4. Дождитесь завершения загрузки или момента, когда она снова зависнет
5. Скопируйте содержимое консоли, отправьте его в [**Discord**](https://discord.gg/PBvG8D9MxT) с описанием проблемы
